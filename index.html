Summary: Latitude vs. X

The purpose of this project was to analyze how the weather changes as you get closer and closer to the equator. To accomplish this analysis, we retrieved data from the OpenWeatherMap API to compile a dataset of over 500 cities.

After assembling the dataset, Matplotlib was used to plot various aspects of the Weather vs. Latitude. Factors we looked at included: Temperature, Cloudiness, Wind Speed, and Humidity. This site provides the source data and various visualizations created as a part of the analysis, as well as explanations and descriptions of any trends and correlations observed.
